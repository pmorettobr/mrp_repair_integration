# 🔗 Integração Reparo - MRP para Odoo 16 Community

Módulo que conecta os fluxos de **Ordens de Reparo** e **Ordens de Produção**, com foco em **experiência do usuário**.

## ✨ Funcionalidades

### 🔗 Vínculo Bidirecional
- Link Repair ↔ MRP com smart buttons
- Navegação rápida entre documentos relacionados
- Notificações automáticas no chatter ao vincular

### 🧙 Wizard Inteligente
- Interface guiada para criar MO a partir de um Reparo
- Auto-preenchimento de dados (produto, quantidade, BOM)
- Validações em tempo real com mensagens amigáveis

### 👤 Gestão de Operador
- Atribuição de técnico responsável em Repair e Work Orders
- Auto-preenchimento baseado no usuário logado
- Registro automático de tempo real (início/fim)

### 🎨 UX Otimizada
- Badges coloridos por status para leitura rápida
- Campos organizados em grupos lógicos com ícones
- Help text contextual em todos os campos customizados
- Filtros personalizados: "Meus Reparos", "Minhas Operações"
